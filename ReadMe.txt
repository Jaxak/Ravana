Равана — правнук Брахмы, внук великого Пуластьи, сын Вишравов. 
Был повелителем ракшасов и мифическим владыкой острова Ланка, персонаж эпоса «Рамаяна», антагонист главного героя — Рамы. 
Могучий гигант имел десять лиц, двадцать рук, меднокрасные глаза, высоченную грудь и зубы с белизной молодой луны.
			
ПАРТНЕРКА:

1. CTRl + 1	
заполняет поля ИНН и название ИП (запоминается в буфере)
2. CTRl + 2 	
заполняет поля ИНН-КПП и название организации (запоминается в буфере)
3. CTRl + 3		
заполняет поля при добавлении контактного лица для организации
4. CTRl + 4
заполняет реквизиты 
(удаляем регион, нажимаем кнопки)
(после закрытия окна, кликаем на поле ФАКТИЧЕСКИЙ АДРЕС)
5. CTRl + 5
заполняем поля при создании нового мэнеджера
6. CTRl + 6
заполнение полей при печате юр документов
(при активном CapsLock заполняет в верхнем регистре)


ПОЛЕЗНЫЕ:

1. CTRl + numpad7	
генератор ФИО в одну строку
2. CTRl + numpad8	
генератор телефона без 8 и +7	
3. CTRl + numpad9	
генератор Email
4. CTRl + Q			
Генерирует ИНН, вставляет, запоминает в буфере
5. CTRl + CapsLock
Переводит выделеный текст в верхний регистр
6. AppKEY - открыть логи
		
			
			CTRl + AppKEY 
			Вызов справки
